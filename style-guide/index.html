Section 08

FAQ Accordion — live demo

Handled automatically by parentwise-design-system.js. No extra JavaScript needed on the page.

Wrap items in .pw-faq, each question/answer pair in .pw-faq-item, the trigger in button.pw-faq-q, and the answer in div.pw-faq-a. Add a span.pw-faq-chevron inside the button for the animated arrow.
No. The design system JS file (/style-guide/parentwise-design-system.js) already handles the accordion. As long as that script is included at the bottom of the page, it works automatically.
Yes. Each .pw-faq container is independent — collapsing one item only affects siblings in the same container.
Show HTML markup 
<div class="pw-faq">
  <div class="pw-faq-item">
    <button class="pw-faq-q" aria-expanded="false">
      Question text
      <span class="pw-faq-chevron" aria-hidden="true">▾</span>
    </button>
    <div class="pw-faq-a">Answer text.</div>
  </div>
</div>

Section 10

Scroll Reveal

Add .pw-reveal to any element. It fades + slides up as it enters the viewport. Grid children automatically stagger (up to 4 items). No JS setup required — handled by the design system script.
